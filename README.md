# CI/CD Learning Labs

A hands-on, from-scratch exploration of GitHub Actions and CI/CD — one
concept at a time, each backed by a working workflow and a note on how
it applies to a real production pipeline.

This isn't a tutorial copy-paste repo. Every workflow here was built to
practice a specific concept, then documented with its real-world use
case (see [`NOTES.md`](./NOTES.md) for the full breakdown).

## Why this repo exists

A public record of hands-on CI/CD skill-building — built to demonstrate
working knowledge of GitHub Actions, with each workflow documented
against a real-world use case.

## Topics covered so far

- Workflow syntax, triggers (`push`, `pull_request`, `workflow_dispatch`)

_(this list grows as new lessons are added — see NOTES.md for details)_

## Structure

```
.github/workflows/   → one workflow file per lesson, numbered in order
NOTES.md             → lesson-by-lesson log: what I learned + real-world application
```
